Hebrew / RTL Implementation Rules
dir="rtl" on every Hebrew container
text-align: right for all Hebrew body copy
Line-height 1.8–2.0 for body text (Hebrew glyphs are taller)
letter-spacing: 0 — always, Hebrew doesn't kern like Latin
Never use Alef below weight 400 — no light variant in Hebrew
Mirror icon positions: left-aligned icons become right-aligned in RTL
Nav link order reverses in RTL — test both directions
unicode-bidi: embed for mixed-direction paragraphs

08 — Implementation
Dev Handoff
Drop-in snippets for web, React, and Flutter. Hebrew RTL support baked into every pattern.
Font import — Playfair Display + DM Sans + Alef
<link href="https://fonts.googleapis.com/css2? family=Playfair+Display:ital,wght@0,400;0,700;1,400 &family=DM+Sans:wght@300;400;500 &family=Alef:wght@400;700 &display=swap" rel="stylesheet"/>
CSS — Hebrew utility class
/* Drop on any Hebrew container */ .he { font-family: 'Alef', Arial, sans-serif; direction: rtl; unicode-bidi: embed; text-align: right; letter-spacing: 0; line-height: 1.9; } .he h1, .he h2, .he h3 { font-weight: 700; } .he p, .he li, .he span { font-weight: 400; } /* RTL button group */ .btn-group[dir="rtl"] { flex-direction: row-reverse; } /* tokens.css already includes: */ /* --font-hebrew: 'Alef', Arial, sans-serif; */
React — bilingual text component
const BravaText = ({ children, lang = 'en', variant = 'body' }) => { const isHe = lang === 'he'; const styles = { fontFamily: isHe ? "'Alef', sans-serif" : "'DM Sans', sans-serif", direction: isHe ? 'rtl' : 'ltr', textAlign: isHe ? 'right' : 'left', lineHeight: isHe ? 1.9 : 1.7, letterSpacing: isHe ? 0 : undefined, fontWeight: isHe ? 400 : 300, }; return <p style={styles}>{children}</p>; }; // Usage: // <BravaText lang="he">שלום עולם</BravaText> // <BravaText lang="en">Hello world</BravaText>
